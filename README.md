# Economy-Engine

Gas Tank model, Ledger, and Meter for cost-aware platform operations.

## Overview

Economy-Engine implements the economic subsystem powering the Vindicta Platform's free-tier sustainability.

## Features

- **Gas Tank**: Credit-based usage tracking
- **Atomic Ledger**: Double-entry transaction log
- **Meter**: Usage metering with pricing multipliers

## Installation

```bash
uv pip install git+https://github.com/vindicta-platform/Economy-Engine.git
```
